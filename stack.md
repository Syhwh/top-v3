# Stack

Write a constructor function called `Stack` with no arguments. It should initialize two attributes:

* `head` with `null`
* `size` with `0`

The methods that you will implement are:

* `push`
* `pop`
* `size`

```javascript
var stack = new Stack();
stack.push(5);
stack.push(8);
stack.pop(); // 8
stack.pop(); // 5
stack.pop(); // null
stack.length(); // 0
```
