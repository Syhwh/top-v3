# TicTacToe

Write a function called `tictactoe` that receives a matrix and returns true if it's winning for 'X' or 'O'.

```javascript
console.log(tictactoe([
  ['X', '', 'O'],
  ['', 'X', 'O'],
  ['O', '', 'X']
])); // true

console.log(tictactoe([
  ['O', 'X', 'O'],
  ['', 'X', 'O'],
  ['O', '', 'X']
])); // false
```
